• Refresh token every 5 minutes:

    Automatically renews the authentication token every 5 minutes to maintain the session active without requiring user interaction. Using interval of RxJS

  • You can refresh the page and the session will not be lost:

    The application SPA preserves user authentication across page reloads, ensuring that the session remains active and data is not lost.

  • Controlled routes:

    Routes are protected using authentication guards to restrict access based on user roles, ensuring that only authorized users can access certain pages.

    Note: Non-admin users are prevented from accessing the Admin User route and are redirected to the forbidden page if they attempt to do so.

  • User CRUD components:

    Includes components for creating, reading, updating, and deleting user data, providing full control over user management within the application.

  • Use of Tailwind:

    Tailwind CSS is used throughout the project for styling, enabling rapid design customization with utility-first classes and responsive design capabilities.

  • Search by name or email:

    Users can be filtered by name or email to quickly find specific user data within the application.

  • Pagination:

    The application supports pagination for user listings using query parameters, allowing efficient data management and display.
